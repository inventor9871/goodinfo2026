# ETF 資訊查詢系統

這是一個讀取既有 `StockList.csv` 並將其視覺化為網頁儀表板的專案。採用 Vanilla HTML/CSS/JS 開發，不需要建立額外的後端資料庫與伺服器，直接在前端進行資料解析、轉換與呈現。專案採用了高質感的暗黑玻璃擬物風格（Glassmorphism），提供現代化且吸睛的操作體驗。

## 專案功能與開發歷程

### 第一階段：基礎單筆查詢建立
* **使用者需求**：需要根據 `StockList.csv` 製作一個查詢網頁，僅關注 **[代號、名稱、成交價、除息交易日、合計股利、年化合計殖利率]** 這 6 項特定資訊。並提供輸入欄位，可以透過輸入代號來馬上查詢該 ETF 資訊。
* **實作細節**：
    1. 建立 `index.html`（結構）、`style.css`（高質感樣式）、`app.js`（核心邏輯）。
    2. 開發輕量級的 CSV 解析邏輯，將原本包含於 Excel 保留格式（如 `="0050"` 取為 `0050` ）及時間前綴標記（如 `'26/01/22` 轉為 `2026/01/22`）自動清理乾淨。
    3. 實作查詢演算法及狀態切換機制（包含載入中、錯誤提示、查無資料防呆 等）。

### 第二階段：全覽列表與 AI 趨勢導入
* **使用者需求**：修改首頁行為，系統一開始須優先「顯示全部的資料」。若有輸入特定代號查詢（或點選資料）時，才單獨顯示該筆特定的查閱卡片，同時要求能「提供簡短未來趨勢的分析」。
* **實作細節**：
    1. **全資料表呈現（List View）**：新增含固定表頭（Sticky Header）的滾動表格元件。將所有清出的 CSV 資料全覽顯示，高殖利率（>= 5%）的數值特別標色提醒。
    2. **互動體驗升級**：不僅可透過上方的搜尋欄位查詢，我也綁定了表格每一列（row），只要「點擊該檔 ETF」即可直接進入個股分析卡片。
    3. **AI 趨勢分析功能**：在查閱特定 ETF 時，系統會根據該檔 ETF 取出的「年化殖利率」、數值區間，甚至判斷是否為槓桿/反向型 ETF，自動渲染出一套簡略實用的未來趨勢配置預測與建議。
    4. 加入「顯示全部 (Back)」功能，提供流暢的使用者返回迴圈。
